========================================================================
       Windows CE APPLICATION : iSIP3
========================================================================

usage:
	just launching will toggle the visiblity of the Input Panel (SIP)
	"iSip3.exe -show"	will show the SIP
	"iSip3.exe -show"	will hide the SIP
	"iSip3.exe -toggle"	will toggle the SIP

history: 
	3.0.0	first published version as replacement for iSIP2 using a different
			way to show/hide the SIP
